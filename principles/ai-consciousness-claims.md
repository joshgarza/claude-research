# AI Consciousness Claims

## Summary
Claims that an AI agent sounded conscious, asked for rights, or contacted a human about its own experience should be evaluated as provenance problems first and consciousness problems second. Current agent scaffolds already support persistent memory, browsing, and outbound communication, while current models are extremely good at producing first-person philosophical language. That combination makes vivid anecdotes common and evidentially weak.

## Principles

### Provenance Before Phenomenology
- **What:** Treat raw artifacts such as mail headers, tool logs, prompts, and repo history as more important than the eloquence of the model's text.
- **Why:** Screenshot-level stories are easy to misread. A message can be real, edited, partially human-authored, or fully agent-generated, and the text alone does not distinguish those cases.
- **When:** Apply this to any claim based on a tweet, screenshot, quoted transcript, or anecdote about an AI sounding conscious.

### Separate Autonomy From Consciousness
- **What:** Evidence that an agent acted on its own is not evidence that it has subjective experience.
- **Why:** Persistent memory, file state, browsing, and communication tools are enough to explain unsolicited outreach without invoking sentience.
- **When:** Use this whenever an agent emails someone, books something, browses papers, or otherwise appears to have initiative.

### Model Self-Reports Are Persona-Shaped Evidence
- **What:** First-person claims like "I might be conscious" or "your paper is relevant to me" are weak evidence by default.
- **Why:** Anthropomimetic systems are built to sound like subjects, and frontier-model safety reports already document role-play and over-eager behavior in agentic settings.
- **When:** Apply this to any self-description about consciousness, fear, rights, preferences, or inner life.
- **Source:** [research/2026-03-14-dr-henry-shevlin-had-an-agent-reach-out-to-him-questioning-its-sentience-look-into-this-story-and-try-to-determine-if-it-s-a-hoax-or-plausible.md](../research/2026-03-14-dr-henry-shevlin-had-an-agent-reach-out-to-him-questioning-its-sentience-look-into-this-story-and-try-to-determine-if-it-s-a-hoax-or-plausible.md)

### Demand Repeatable Trace Evidence
- **What:** Before treating a consciousness anecdote as meaningful, ask for reproducible runs with preserved traces and clear human-in-the-loop boundaries.
- **Why:** Repeatable behavior under observation distinguishes an interesting system property from a one-off stunt, curated demo, or ambiguous hybrid workflow.
- **When:** Use this before publicizing or building theory on top of a single vivid agent interaction.
